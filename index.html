What is Terraform?


Terraform belongs to the provisioning part of the DevOps tools and it is an open-source infrastructure as code software tool created by HashiCorp. It is quietly revolutionizing DevOps by changing the way infrastructure is managed, and making it faster and more efficient to execute DevOps projects. It's one type of tech tool that lets you build, change, and version infrastructure safely and efficiently. This tool abstracts out the interaction with various infrastructure services (AWS, Digital Ocean, OpenStack, etc) and provides a unified configuration format for it.
